New Relic のモバイルモニタリングの UI には、モバイルアプリインデックスがあり、モニターしているアプリのリストとそのアプリに関する重要なサマリー情報が表示されます。

## モバイルアプリの一覧を見る [#menu]

モバイルアプリケーションのインデックスを見るには、\[**[ one.newrelic.com](https://one.newrelic.com) > エクスプローラー> モバイルアプリケーション** をご利用ください。 [エクスプローラー](/docs/new-relic-one/use-new-relic-one/core-concepts/new-relic-explorer-view-performance-across-apps-services-hosts/) を使用して、すべての [エンティティ](/docs/new-relic-one/use-new-relic-one/core-concepts/what-entity-new-relic/) 、つまり、アプリケーションやホストから任意の要素のカスタムグルーピングまで、データを報告する特定できるものすべてにアクセスできます。

または、 **[one.newrelic.com](https://one.newrelic.com) > Mobile**.

利用可能なモバイルアプリのインデックスには、色付きの健康状態を示すものが含まれています。

* グリーン＝ノーマル
* 黄＝警告
* 赤＝クリティカル
* グレー = データを報告していない

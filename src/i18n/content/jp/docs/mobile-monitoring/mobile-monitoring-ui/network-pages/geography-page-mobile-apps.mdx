Geographyページでは、モバイルユーザーの体験を世界観として表示しています。

* 応答時間をカラーで表示
* ネットワークリクエスト（1分あたりの通話数）
* データ転送サイズ
* アクティブデバイス
* ネットワーク故障率

また、各国の詳細情報にドリルダウンすることもできます。

<Callout variant="tip">
  **Geography** 機能は、 **Map** 機能とは異なります。 [**Map** page](/docs/mobile-monitoring/mobile-monitoring-ui/network-pages/map-page-mobile-apps) は、モバイルアプリとその関連サービスの関係をアーキテクチャ的に示しています。
</Callout>

## Geographyページの表示 [#geo_procedures]

携帯電話の応答時間を国別に表示したり、並べ替えることができます。

1. Go to **[one.newrelic.com](https://one.newrelic.com) > Mobile> （アプリを選択）> Network> Geography**.
2. 表示される情報（応答時間、1分あたりのリクエスト数、総転送サイズ、アクティブなデバイス、ネットワーク障害率など）を変更するには、 **「Sort by」** メニューから選択します。
3. 表示される情報の量を調整するには、 **Hide&lt; 1% throughput** を選択します。
4. 地図上の色のついた部分にマウスオーバーするか、リスト上の国名にマウスオーバーすると、その場所の概要情報が表示されます。

当社標準の [ユーザーインターフェース機能](/docs/accounts-partnerships/education/getting-started-new-relic/new-relic-user-interface) および [ページ機能](/docs/data-analysis/user-interface-functions/standard-page-functions) を使用して、詳細な情報にドリルダウンすることができます。

## ドリルダウンの詳細表示 [#geo_details]

特定の場所の詳細情報（平均応答時間、1分あたりの通話数、アクティブなデバイス、種類別のネットワーク障害など）を表示するには、地理ページの地図上でその場所を選択するか、リストで名前を選択します。

## 標準インストレーション [#install]

これは、[Pythonエージェントが自動インストゥルメンテーションを行うフレームワーク上のアプリケーション](/docs/agents/python-agent/getting-started/instrumented-python-packages)向けの手順です。

* Web以外のPythonスクリプト、カスタムバックグラウンドタスク、または非自動インストゥルメンテーション フレームワーク上のアプリケーションをモニターする場合は、[Web以外のスクリプト/関数のモニタリング](/docs/agents/python-agent/supported-features/python-background-tasks)に関するドキュメント参照してください。
* Dockerを使用している場合は、各コンテナ内にエージェントをインストールします。

New Relic Pythonエージェントを[自動インストゥルメンテーションフレームワーク上の](/docs/agents/python-agent/getting-started/instrumented-python-packages)アプリケーションにインストールするには：

1. New Relicアカウントをまだお持ちでない場合は、[New Relicアカウントを新たに作成します](https://newrelic.com/signup)。永久無料です。

2. New Relic[ライセンスキー](/docs/subscriptions/license-key)を用意しておきます（New Relicアカウントを保有していない場合は、[サインアップして作成](https://newrelic.com/signup)します）。

3. 以下のように**pip**を実行して、**newrelic**Pythonパッケージをインストールします。

   ```
   pip install newrelic
   ```

4. 以下のコマンドを実行して、エージェント設定ファイルを生成します。

   ```
   newrelic-admin generate-config <var>YOUR_LICENSE_KEY</var> newrelic.ini
   ```

5. ウェブアプリケーションまたはWSGIサーバーの起動を制御する場合は、通常の起動コマンドオプションの前に以下のNew Relic管理者スクリプトコマンドを使用します。

   ```
   NEW_RELIC_CONFIG_FILE=newrelic.ini newrelic-admin run-program <var>YOUR_COMMAND_OPTIONS</var>
   ```

   この方法で管理者スクリプトを実行できない場合は、[高度なインストール手順](/docs/agents/python-agent/installation-configuration/python-agent-installation)を読んで、より詳しいコンテキストおよびその他のインストールオプションを確認します。

6. PythonウェブアプリケーションまたはWSGIサーバーを起動または再起動します。5分待ってから、New Relic UIに移動してデータを確認します。

詳細なインストール手順および有用なコンテキストについては、「[New Relic Pythonエージェントの高度なインストール](/docs/agents/python-agent/installation-configuration/python-agent-installation)」をご覧ください。

## APMおよびインフラストラクチャデータのログを表示 [#logs-context]

また、ログとアプリケーションのデータを統合して、トラブルシューティングをより簡単かつ迅速にできます。[コンテキストのログ](/docs/logs/logs-context/configure-logs-context-python/)では、エラーとトレースに関連するログメッセージをアプリケーションのUIで直接表示できます。また、Kubernetesクラスタなど、[インフラストラクチャデータ](/docs/logs/forward-logs/forward-your-logs-using-infrastructure-agent/)のコンテキストでログを表示することもできます。New Relic Oneで別のUIページに切り替える必要はありません。

## その他のオプション [#other]

標準インストレーションを使用できない場合、またはインストレーション手順の詳細やコンテキストが必要な場合は、「[New Relic Pythonエージェントの高度なインストール](/docs/agents/python-agent/installation-configuration/python-agent-installation)」ドキュメントをご覧ください。

また、Pythonエージェントを次の場所にインストールすることもできます。

* [Google App Engineフレキシブル環境](/docs/agents/python-agent/hosting-services/install-new-relic-python-agent-gae-flexible-environment)。
* [AWS Lambda](/docs/serverless-function-monitoring/aws-lambda-monitoring/enable-lambda-monitoring/instrument-example/)

詳しい情報は、「[互換性と要件](/docs/agents/python-agent/getting-started/compatibility-requirements-python-agent)」をご覧ください。

## 構文

```
newrelic_get_browser_timing_footer([bool $include_tags])
```

HTML出力の最後に注入するブラウザモニタリングスニペットを返します。

## 要件

すべてのAgentバージョンに対応しています。

## 説明

HTML 出力の一番最後に注入する JavaScript 文字列を返します。このコールは、 [`newrelic_get_browser_timing_header()`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_header) と一緒に使用して、ウェブページにブラウザモニタリングを手動で追加します。可能であれば、New Relic UI を通じてブラウザモニタリングを有効にするか、JavaScript スニペットをコピー/ペーストすることをお勧めします。これらのオプションの使用方法については、 [Browser monitoring and the PHP agent](/docs/agents/php-agent/features/page-load-timing-php) を参照してください。

<Callout variant="tip">
  [`newrelic_disable_autorum()`](/docs/agents/php-agent/php-agent-api/newrelic_disable_autorum) を比較すると、 **は、ページ上のブラウザ監視スクリプト** を無効にします。
</Callout>

## 戻り値

ブラウザモニタリング用の JavaScript フッター文字列を返します。

## 例

```

    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_header = newrelic_get_browser_timing_header();
    }
    <var>... // Output to page</var>
    if (extension_loaded('newrelic')) { // Ensure PHP agent is available
        $newrelic_footer = newrelic_get_browser_timing_footer();
    }
}
```

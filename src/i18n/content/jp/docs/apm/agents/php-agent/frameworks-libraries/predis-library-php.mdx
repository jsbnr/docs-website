New Relic は PHP agent [リリース 4.22](/docs/release-notes/agent-release-notes/php-release-notes/php-agent-422099) で Predis ライブラリのサポートを追加しました。

エージェントは、 `Predis\Client` オブジェクトまたはパイプラインへの標準的なコールを計測します。パイプライン経由で送信されたコマンドは、個々のコマンドではなく、 `pipeline` という単一のコマンドメトリックとしてグループ化されて扱われます。

PHP エージェントは、 `executeRaw` や `transaction` メソッド、PubSub、monitor などの呼び出しをサポートしていません。これらはインストルメントされません。

## 構文

```
newrelic_get_browser_timing_header([bool $include_tags])
```

HTML出力のヘッドに注入するブラウザモニタリングスニペットを返します。

## 説明

HTML 出力の `< head>` に注入する JavaScript 文字列を返します。このコールは、 [`newrelic_get_browser_timing_footer`](/docs/agents/php-agent/php-agent-api/newrelic_get_browser_timing_footer) と一緒に使用して、ウェブページにブラウザモニタリングを手動で追加します。可能であれば、New Relic UI を使ってブラウザモニタリングを有効にするか、JavaScript スニペットをコピー/ペーストすることをお勧めします。これらのオプションの使用方法については、 [ブラウザモニタリングとPHP](/docs/agents/php-agent/features/page-load-timing-php) を参照してください。

<Callout variant="tip">
  [`newrelic_disable_autorum`](/docs/agents/php-agent/php-agent-api/newrelic_disable_autorum) を比較すると、 **はページ上のブラウザスクリプト** を無効にしています。
</Callout>

## 戻り値

ブラウザエージェントのJavaScriptヘッダー文字列を返します。

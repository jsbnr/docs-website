    * ストレージ使用率(%)

## アプリケーションとKubernetesの連携 [#link-app]

Kubernetesのメタデータを表面化し、APMエージェントにリンクさせることができます。 `MutatingAdmissionWebhook` は、必要な環境変数を Pods に追加するために使用されます。APMは注入された環境変数を使用して、そのAPMアプリケーションに関連する特定のKubernetesのデプロイメント情報を表示します。

New Relicのメタデータインジェクションは、New RelicのKubernetes統合のインストール時に、 `newrelic-metadata-injection` のラベルを通して自動的に追加されます。詳細については、 [アプリケーションを Kubernetes にリンクする方法](/docs/kubernetes-pixie/kubernetes-integration/advanced-configuration/link-your-applications-kubernetes/) をご覧ください。

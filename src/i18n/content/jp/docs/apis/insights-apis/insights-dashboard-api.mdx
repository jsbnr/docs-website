## エンドオブライフのお知らせ [#eol]

Insights Dashboard APIは2021年に終了しました。

* 2021年7月28日現在、 `CREATE` と `UPDATE` のエンドポイントは利用できません。
* 2021年8月30日現在、 `GET` と `DELETE` のエンドポイントは利用できません。

Insights Dashboard API から New Relic One Dashboards API への移行については、 [移行ガイド](/docs/new-relic-one/use-new-relic-one/core-concepts/dashboards-api-migration-insights-api-nerdgraph/) をご参照ください。詳細については、 [NerdGraph dashboards tutorial](/docs/apis/nerdgraph/examples/create-widgets-dashboards-api/) および [Explorers Hub post](https://discuss.newrelic.com/t/important-insights-dashboard-api-end-of-life/149357) をご覧ください。

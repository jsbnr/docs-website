## 特定のモニターエンドポイント [#specific-monitor]

特定のモニターに対してREST APIコールを行う場合は、エンドポイントの一部として `monitor_uuid` を含めてください。 `monitor_uuid` は、URL の一部である GUID です。たとえば、選択された [合成モニター](/docs/synthetics/new-relic-synthetics/pages/synthetics-overview-page-view-monitors-performance) は、このような URL になります。

```
https://synthetics.newrelic.com/accounts/<var>nnnn</var>/monitors/<var>ab123-c456d-e78-90123-f45g</var>
```

`monitor_uuid` は、 `/monitors/` に続く値です。

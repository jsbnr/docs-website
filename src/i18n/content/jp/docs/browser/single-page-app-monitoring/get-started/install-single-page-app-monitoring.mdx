シングルページアプリ（SPA）の監視には、ブラウザエージェント [がデフォルトでインストールされています。](/docs/browser/single-page-app-monitoring/get-started/spa-compatibility-requirements/)

## [要件 [#requirements]](/docs/browser/single-page-app-monitoring/get-started/spa-compatibility-requirements/)

[](/docs/browser/single-page-app-monitoring/get-started/spa-compatibility-requirements/)[コンパティビリティと SPA モニタリングの要件については、こちら](/docs/browser/single-page-app-monitoring/get-started/spa-compatibility-requirements/) をご覧ください。

New Relic アカウントで最初のモニターアプリを設定する際には、利用規約に同意する必要があります。利用規約に同意することで、New Relic が URL からハッシュフラグメントを収集することを許可します。チェックボックスのオプションを選択するのは、1つのアカウントに対して1回だけです。

## SPAモニタリングの有効化・無効化 [#enable-spa]

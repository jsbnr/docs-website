ブラウザエージェントが報告するデータを規制する設定オプションがいくつかあります。これは、 [ブラウザの監視を有効にする際の要因となります。](/docs/browser/browser-monitoring/installation/install-browser-monitoring-agent) また、これらの設定は後から変更することもできます。

アプリの設定画面を表示するには、次のようにします。

1. [one.newrelic.com](http://one.newrelic.com) にアクセスし、 **Browser** をクリックします。
2. アプリを選択します。
3. 左のナビゲーションで、 **アプリの設定** をクリックします。

アクセスできない場合は、パーミッションが原因かもしれません。

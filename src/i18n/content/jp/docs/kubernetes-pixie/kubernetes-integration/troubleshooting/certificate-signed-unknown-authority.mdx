## 問題 [#logs-versions]

**`x509: certificate signed by unknown authority`** というメッセージが表示されます。

## 解決策

署名付き証明書が正しく設定されていることを確認する必要があります。そのためには、マニフェストの `DaemonSet` 部分で以下の変数を使用して、 `.pem` ファイルを設定します。

```
- name: NRIA_CA_BUNDLE_DIR 
  value: <var>YOUR_CA_BUNDLE_DIR</var> 
- name: NRIA_CA_BUNDLE_FILE  
  value: <var>YOUR_CA_BUNDLE_NAME</var>
```

_YAMLキーパス: `spec.template.spec.containers.name.env`_

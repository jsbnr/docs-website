[Elixir](https://github.com/newrelic/elixir_agent) の動作を、Elixir のオープンソースエージェントを使って New Relic で監視します。エージェントが

* トランザクション、分散型トレース、およびアプリケーションの動作の他の部分を追跡するのに役立ちます。
* 基礎となる [BEAM](http://erlang.org/faq/implementations.html#idp32950544) 活動の概要を提供します。

<Callout variant="tip">
  このエージェントは、オープンソースとして [GitHub](https://github.com/newrelic/) で公開されています。また、最新の更新情報としてチェンジログも公開されています。
</Callout>

## スタートガイド [#get_started]

要件、インストール、および設定に関する情報は、GitHubに掲載されている [Open Source Elixir Agent README](https://github.com/newrelic/elixir_agent/blob/master/README.md) をご覧ください。

[New Relic's Elixir repository on GitHub](https://discuss.newrelic.com/t/introducing-new-relic-s-open-source-elixir-agent/58566) インストール、使用方法、その他のトピックに関する質問はこちらをご覧ください。問題やバグは、 [GitHub リポジトリ](https://github.com/newrelic/elixir_agent/issues) の Issue として報告してください。

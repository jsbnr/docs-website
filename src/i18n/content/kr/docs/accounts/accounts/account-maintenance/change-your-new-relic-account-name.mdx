## 사용자 이름 변경 [#user-name]

사용자 이름을 변경하려면:

1. [계정 드롭다운](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#account-dropdown) 에서 **사용자 기본 설정** 을 선택합니다.
2. 오른쪽에서 **전체 이름** 을 클릭합니다.

## 계정 이름 변경 [#account-name]

[원래 사용자 모델](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) 의 사용자만 New Relic 계정 이름을 변경할 수 있습니다. 이것을하기 위해:

1. [계정 드롭다운](/docs/accounts-partnerships/education/getting-started-new-relic/glossary#account-dropdown) 에서 **계정 설정** 을 선택합니다.
2. 오른쪽에서 **이름** 필드를 편집합니다.

<Callout variant="important">
  [New Relic One 사용자 모델](/docs/accounts/original-accounts-billing/original-users-roles/overview-user-models) 을 사용 중이고 계정 이름을 변경하려면 계정 담당자나 [지원팀](/docs/using-new-relic/welcome-new-relic/get-started/find-help-use-support-portal#file-ticket) 에 문의하세요.
</Callout>

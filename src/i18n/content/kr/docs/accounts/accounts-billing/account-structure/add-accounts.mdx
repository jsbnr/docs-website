title: 계정 추가
metaDescription: How to add accounts to your New Relic organization.
translationType: machine
---

New Relic 조직에 계정을 추가하는 방법은 사용자가 속한 [사용자 모델](/docs/accounts/original-accounts-billing/original-product-based-pricing/overview-user-models) 유형에 따라 다릅니다.

* New Relic One 사용자 모델: 최신 사용자 모델을 사용 중인 경우 현재 UI를 사용하여 계정을 추가할 수 없지만 계정 담당자에게 문의해야 합니다. 자세한 내용은 [조직 구조](/docs/accounts/accounts-billing/account-structure/new-relic-account-structure) 를 참조하십시오.
* 원래 사용자 모델: UI를 통해 계정을 추가할 수 있습니다. 자세한 내용은 [계정 추가](/docs/accounts/original-accounts-billing/original-users-roles/parent-child-account-structure/#creating) 를 참조하십시오.

현재 사용 중인 항목을 확인하고 싶으신가요? [사용자 모델 결정](/docs/accounts/original-accounts-billing/original-users-roles/overview-user-models/#determine-user-model) 을 참조하십시오.

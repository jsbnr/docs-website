## 밝은 모드와 어두운 모드 간 전환 [#dark-mode]

현재 Firefox 브라우저에서는 다크 모드가 지원되지 않습니다. 그렇지 않으면 어두운 모드에서 UI를 보려면:

1. **[one.newrelic.com](https://one.newrelic.com)** 에서 [계정 드롭다운](/docs/using-new-relic/welcome-new-relic/get-started/glossary#account-dropdown) 을 클릭합니다.

2. **테마:** 를 클릭하여 다음 옵션 중 하나를 선택합니다.

   * **테마: auto** (기본값): UI를 OS에서 이미 활성화한 것과 일치시킵니다.
   * **테마: dark** : UI를 어두운 모드로 유지합니다.
   * **테마: light** : UI를 조명 모드로 유지합니다.

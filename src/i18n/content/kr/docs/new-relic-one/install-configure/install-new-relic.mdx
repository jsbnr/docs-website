## 문제점 해결

에이전트를 설치하고 약간의 트래픽을 생성하고 몇 분 정도 기다린 후에 New Relic UI에서 데이터를 보기 시작해야 합니다. 데이터가 표시되지 않으면 [문제 해결 절차](/docs/agents/manage-apm-agents/troubleshooting/not-seeing-data) 를 따르십시오.

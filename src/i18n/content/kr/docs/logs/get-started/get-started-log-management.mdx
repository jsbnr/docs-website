당사의 로그 관리 솔루션은 애플리케이션 및 인프라 성능 데이터(이벤트 및 오류)에 대한 심층적인 가시성을 제공하여 MTTR(평균 해결 시간)을 줄이고 프로덕션 인시던트 문제를 신속하게 해결합니다.

## 문제를 더 빨리 찾고 컨텍스트 전환을 줄입니다. [#logs-definition]

로그 관리는 로그 데이터를 나머지 애플리케이션 및 인프라 데이터와 연결하는 방법을 제공합니다. 도구 간의 컨텍스트 전환을 유지하면서 문제의 근본 원인을 빠르게 찾을 수 있습니다.

로그 관리 기능은 다음과 같습니다.

* 로그를 즉시 검색합니다.
* 로그 UI에서 직접 로그 데이터를 시각화하세요.
* 로깅 데이터를 사용하여 맞춤형 차트, 대시보드 및 경고를 생성합니다.
* 도구 간에 전환하지 않고 성능 문제를 해결합니다.
* 한 곳에서 모든 것을 시각화하십시오.

## 요구 사항 [#requirements]

로그 관리 기능에 대한 액세스는 사용자 유형에 따라 제한됩니다. 자세한 내용은 [사용자 유형 기능](/docs/accounts/accounts-billing/new-relic-one-user-management/user-type#user-type-capabilities) 을 참조하십시오.

## 로깅 데이터 가져오기 [#integrate-logs]

로그 데이터를 New Relic에 전달하려면 다음을 수행할 수 있습니다.

* 추가 소프트웨어를 설치할 필요 없이 인프라 모니터링 에이전트를 경량 데이터 수집기로 사용하십시오.
* Amazon, Microsoft, Fluentd, Fluent Bit, Kubernetes, Logstash 등을 포함한 다양한 로그 전달 플러그인 중에서 선택하십시오.
* OpenTelemetry 솔루션을 사용하십시오.
* Log API 또는 TCP 엔드포인트를 사용하여 로그 데이터를 보냅니다.

로그 관리가 활성화되면 로그를 APM 에이전트, Kubernetes 클러스터 또는 분산 추적과 연결하여 컨텍스트 확장의 로그를 사용하여 추가 컨텍스트 로깅 데이터를 얻을 수도 있습니다.

## New Relic에서 로깅 데이터 보기 [#find-data]

UI 또는 API로 로깅 데이터를 탐색할 수 있습니다.

* [one.newrelic.com](https://one.newrelic.com) 의 로그 UI
* 해당하는 경우 EU 지역 데이터 센터의 로그 UI: [one.eu.newrelic.com](https://one.eu.newrelic.com)

`Log` 데이터 유형을 쿼리할 수도 있습니다. 예를 들어 NRQL을 사용하여 다음을 실행합니다.

```
SELECT * FROM Log
```

또한 GraphQL 형식 API인 NerdGraph를 사용하여 필요한 정확한 데이터를 요청할 수 있습니다.

다음은 앱의 관련 엔터티(이 예에서는 워크로드 서비스) 중 하나에 대한 로그 데이터의 예입니다.

## 무엇 향후 계획 [#what-next]

로그 관리 솔루션을 시작할 준비가 되셨습니까?

1. 아직 계정이 없다면 [New Relic 계정을 만드](https://newrelic.com/signup) 십시오. 영원히 무료입니다.
2. 로그를 New Relic에 [전달하여](/docs/logs/forward-logs/enable-log-management-new-relic/) 로그 관리를 활성화하십시오. **권장 사항:** 당사의 [인프라 에이전트](/docs/logs/forward-logs/forward-your-logs-using-infrastructure-agent/) 를 로그 전달자로 사용하여 당사 UI에서 직접 플랫폼 및 서비스의 컨텍스트에서 로그를 얻을 수 있습니다.
3. New Relic APM 에이전트가 모니터링하는 앱의 경우 [컨텍스트 에서 로그를](/docs/logs/logs-context/configure-logs-context-apm-agents/) 구성합니다.
4. 경고를 추가하고, 데이터를 쿼리하고, 대시보드를 생성할 수 있는 New Relic One의 [로그 UI](/docs/logs/ui-data/use-logs-ui/) 를 사용하여 플랫폼 전체에서 로깅 데이터를 탐색하십시오.

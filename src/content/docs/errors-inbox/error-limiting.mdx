## Problem [#problem]

A limited number of accounts may receive the following warning when attempting to use Errors inbox:

```
Daily limit on number of unique error groups exceeded.
```

When this limit is reached, Errors inbox will no longer update and errors will seem to plateau or fall off.

## Solution [#solution]

To solve this issue, [reduce the scope of your workloads](/docs/new-relic-one/use-new-relic-one/workloads/use-workloads/).
